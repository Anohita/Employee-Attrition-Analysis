# Employee-Attrition-Analysis
Data analysis and ML project to predict employee attrition and provide retention insights.

## Problem Statement
Employee attrition is a significant challenge for organizations, leading to increased recruitment costs, loss of experienced talent, and reduced productivity.
This project analyzes employee demographic,compensation, and work-related factors to identify key drivers of attrition and builds a predictive model to identify employees at risk of leaving.

## Objectives 
- Perform exploratory data analysis to undertstand attrition patterns.
- Identify key factors influencing employee attrition
- Build a ML model to predict employee attrition
- Provide data-driven recommendations to improve employee retention
- Estimate potential business impact through reduced attrition

## Dataset 

**IBM HR Analytics Employee Attrition & Performance**

Source: Kaggle
This dataset contains fictional employee data including age, job role, income, job satisfaction, years at company, and attrition status.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn


## Key Outcomes 
- Identification of high-risk employee groups
- Predictive model for attrition risk
- Actionable retention strategies
